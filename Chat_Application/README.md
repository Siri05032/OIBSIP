# 💬 Advanced GUI Chat Application

A modern GUI-based Chat Application developed using **Python Tkinter** with multiple chat rooms, chatbot mode, image sharing, notifications, and chat history features.

---

# 🚀 Features

- 🔐 User Login System
- 💬 Real-Time Chat Interface
- 👥 Multi-User Chat Support
- 🤖 Chatbot Mode
- 😀 Emoji Support
- 🏠 Multiple Chat Rooms
- 📷 Image Sharing
- 🕒 Message Timestamps
- 🎨 Modern Dark GUI Design
- 🔄 User Switching Feature
- 📂 File Handling Support

---

# 🛠️ Technologies Used

- Python
- Tkinter
- File Handling
- Datetime Module
- Emoji Library

# ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/OIBSIP.git
```

2. Open Chat_Application folder

```bash
cd Chat_Application
```

3. Install required library

```bash
pip install emoji
```

4. Run the project

```bash
python chat_application
